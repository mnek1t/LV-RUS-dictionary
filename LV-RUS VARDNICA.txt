Программа для запоминания слов на латышском
--------------------------------------------------------------------------------------------------------------------
Суть приложения состоит в том чтобы выводить на экран слово на русском языке, а пользователь вводил в текстовое поле
перевод на латышский и программа давала ответ Тру или Фолс
--------------------------------------------------------------------------------------------------------------------
Создать WPF приложение
Файлы в решении: Dictionary(Model), Dictionary(Presenter), README, ДИЗАЙН, WPFfile.xaml, WPFfile.xaml.cs, Program.cs
--------------------------------------------------------------------------------------------------------------------
1. Создать файл в котором будут записаны слова на русском (Dictionary <string, string>)
2. Создать файл в котором будут записаны слова на латышском
Файл будет в формате txt или json как в приложении с BankAccount
3. соответсвенно добавить первый файл в Dictionary Keys, а второй в Dictionary Values
4. В нашем Dictionary мы доожны уметь добавлять слово и делать проверку существует ли уже такой ключ.
5. Как доп. добавить возмодность удалять слова из словаря
--------------------------------------------------------------------------------------------------------------------
Библиотеки и классы необходимые для использования:
Dictionary <TKey, TValue>
StreamWriter
FileStream
Если файл будет json необходимо будет добавить NuGet и тд в соответсии с видео Пишем список дел WPF от Simplecode
